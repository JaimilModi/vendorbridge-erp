# VendorBridge ERP

VendorBridge is a Procurement & Vendor Management ERP.

## Technology Stack

- **Frontend**: React, Vite, TypeScript, Ant Design, React Router, Axios, Recharts
- **Backend**: Node.js, Express, TypeScript, Prisma ORM, JWT, bcrypt
- **Database**: PostgreSQL (Neon)

## Folder Structure

- `frontend/`: Vite configuration, React layouts, services, hooks, types, and analytics dashboards.
- `backend/`: Express app setup, database config, middleware, controllers, routes, and services.
- `docs/`: Development directives, architectural rules, and context logs.
