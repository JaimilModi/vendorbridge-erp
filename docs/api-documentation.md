# API Documentation

The VendorBridge REST API provides 42 robust endpoints for complete procurement operations. All responses adhere to a uniform JSON envelope.

## Response Formats

**Success Example:**
```json
{
  "success": true,
  "message": "Quotation submitted successfully.",
  "data": { ... }
}
```

**Error Example:**
```json
{
  "success": false,
  "message": "Validation failed.",
  "code": "VALIDATION_ERROR",
  "errors": [ { "field": "email", "message": "Must be a valid email" } ]
}
```

## Endpoints Summary

### Auth
- `POST /api/auth/signup`: Create a new user (assigns role).
- `POST /api/auth/login`: Authenticate and receive a JWT.
- `GET /api/auth/me`: Get current authenticated user details.

### Vendors
- `GET /api/vendors`: List vendors with optional filters.
- `POST /api/vendors`: Register a vendor profile.
- `PATCH /api/vendors/:id/status`: Update vendor onboarding status.

### RFQs
- `POST /api/rfqs`: Create a new draft RFQ.
- `PATCH /api/rfqs/:id/publish`: Move RFQ to published state.
- `POST /api/rfqs/:id/vendors`: Assign invited vendors to an RFQ.

### Quotations
- `GET /api/quotations/rfq/:rfqId`: Fetch the comparison matrix for an RFQ.
- `POST /api/quotations`: Save a vendor's draft quotation.
- `PATCH /api/quotations/:id/submit`: Move quotation to submitted status.
- `PATCH /api/quotations/:id/select`: Mark a quotation as winner (rejects others).

### Approvals
- `POST /api/approvals`: Request internal manager approval.
- `PATCH /api/approvals/:id/decide`: Manager approves/rejects (triggers PO generation).

### Purchase Orders
- `GET /api/purchase-orders`: View generated POs.
- `PATCH /api/purchase-orders/:id/status`: Mark PO as fulfilled/acknowledged.

### Invoices
- `POST /api/invoices`: Vendor issues an invoice against an issued PO.
- `PATCH /api/invoices/:id/status`: Mark invoice as paid or rejected.

### Activity Logs & Reports
- `GET /api/activity-logs`: Global system audit trail.
- `GET /api/reports/summary`: Dashboard KPI data.
- `GET /api/reports/vendor-performance`: Win-rate and reliability metrics.
